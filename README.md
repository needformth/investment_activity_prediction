В данном файле представлен прогноз инвестиционной активности населения.
Прогноз строится с помощью Ridge регрессии. Такой выбор обусловлен результатами валидации, а также особенностью предикторов: каждый имеет влияние.
Целевая переменная: Y – вовлеченность трудовых ресурсов в инвестиционную деятельность = кол-во физ лиц на Мосбирже / занятое население

X1 – Относительный прирост реальной среднемесячной ЗП, %

X2 – Цена нефти марки Brent на конец года, долл. США за баррель

X3 – Среднегодовой курс доллара, руб. за долл.

X4 – Экспорт, выраженный в долях от ВВП РФ, %

X5 – Инфляция в %, декабрь к декабрю

X6 – Ключевая ставка в среднем за год, %

X7 – Темп прироста М2, %

X8 – Уровень безработицы, %

Данные приводятся к стационарному виду, валидация моделей происходит с помощью метода расширяющегося окна Expanding Window, реализованного самостоятельно.
Оценка происходит по параметру RMSE, также для оценки высчитывается MAPE. В работе также проводится тест на причинность Грейнджера и применяется авторегрессия
